# TECS仕様書(tecs-0-md)-複合セルとは何か

複合セルは、**複合セルタイプから生成されるセル**である。複合セルタイプは複数のセルを組み合わせて複合機能を備えた新しいセルタイプを生成するものであり、その内部のセルを**内部セル**、複合セルタイプから生成されるセルを**複合セル**と呼ぶ。

---

## 複合セルタイプの構成

複合セルタイプは以下を持つ：

| 要素 | 説明 |
|------|------|
| 属性 | 外部から設定可能な定数値。内部セルに結合 |
| 呼び口 | 外部公開の呼び口。内部セルの呼び口または受け口に結合 |
| 受け口 | 外部公開の受け口。内部セルの受け口に接続 |
| 内部セル | 複合セルを構成するセルのインスタンス |

複合セルタイプに属するセルは、複合セルタイプを構成するセルのコピーとして展開される。

## シングルトンとアクティブの伝播

| 条件 | 結果 |
|------|------|
| 内部セルのいずれかがシングルトンセルタイプに属する | 複合セルタイプもシングルトン |
| 内部セルのいずれかがアクティブセルタイプに属する | 複合セルタイプもアクティブ |

## 外部結合と内部結合

- **外部結合**: 複合セルタイプの外部から、その複合セルの呼び口を別のセルの受け口に結合する
- **内部結合**: 複合セルタイプ内部で、内部セル間または内部セルと外部口の間を結合する
